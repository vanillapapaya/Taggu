## 별도 사업 — SaaS 비전 (장기, 독립 프로젝트 가능성)

원래 docs/ 폴더가 이 비전을 가지고 시작됨. 현재 로컬 도구는 일종의 PoC + 트래픽/신뢰 채널 역할.

### 가설
> 한국어 사용자가 짤을 자연어로 검색하고, 발견된 짤의 원작자/시리즈로 연결되는 플랫폼.

### 차별 포지션
- 검색 강한 곳(SauceNAO 등)은 피드 없음
- 피드 강한 곳(9GAG, iFunny)은 검색 약함
- 한국어 + 검색 + 피드 + 창작자 연결을 동시에 갖춘 서비스 부재

### 핵심 기능 (3개)
1. **짤 검색** (유입) — 자연어 → 이미지, CLIP/SigLIP + 캡션 하이브리드
2. **짤 피드** (리텐션) — 숏폼 스와이프 UX, 행동 기반 추천
3. **창작자 연결** (확장) — 원작자 프로필, 출처 추적, 커미션 연결

### 수익 모델
- 피드 인터스티셜 광고 (DAU 3,000+ 손익분기)
- 창작자 프로모션
- 프리미엄 프로필 (포트폴리오 + 통계)
- 커미션 거래 수수료

### 현재 로컬 도구의 SaaS 자산
- ✅ 검색 엔진 코어 (CLIP + WD14 + 한국어 태깅)
- ✅ 캐릭터 인식 / 학습 시스템
- ✅ 한국어 UX / 디자인 시스템
- ✅ 4-provider AI 추상화 (운영 비용 유연성)
- ❌ 인증/계정/멀티테넌트
- ❌ 콘텐츠 수집 파이프라인 (크롤/API)
- ❌ 추천 시스템 (협업 필터링 등)
- ❌ 광고/결제 인프라
- ❌ 모더레이션 / 저작권 takedown 절차

이 SaaS는 사실상 새 프로젝트. 현재 로컬 도구는 기술 검증 + 디자인 검증 + 사용자 인터뷰 채널로 가치 큼.

## 의사결정 마디

### v1 → v1.1로 가는 트리거
- 사용자 본인 라이브러리 1000장 돌파 → 검색 속도 한계 체감
- 본인 외 사용자 5명+ → 설치 마찰 데이터 수집
- "GPU 없는 친구도 쓸 수 있어?" 질문 받음 → API 모드 안내 개선

### v2 / SaaS 트리거
- v1.1 사용자 100명+ → 시장 검증
- 광고 외 수익 모델 검증 (특히 Pro 결제 willingness 확인)
- 별도 도메인/회사 vs 개인 프로젝트 선택
