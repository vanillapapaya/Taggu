## 핵심 컴포넌트

### Backend

| 모듈 | 책임 | 주요 의존성 |
|---|---|---|
| `app.py` | FastAPI app factory, 모든 엔드포인트, 학습 hook | fastapi, uvicorn |
| `index.py` | 폴더 스캔, 인덱싱 루프, CLIP 임베딩, backfill, dedupe, relocalize | torch, open_clip, numpy |
| `providers.py` | VLMProvider 추상화, 4종 구현, with_retry, 번역 헬퍼 | (lazy) openai/anthropic/google-genai/transformers |
| `wd14_tagger.py` | WD14 ONNX 로드, 이미지 → 캐릭터/일반 영문 태그 | (lazy) onnxruntime, huggingface_hub |
| `ccip_tagger.py` | CCIP ONNX 로드, 768d 캐릭터 동일성 임베딩 | (lazy) onnxruntime, huggingface_hub |
| `desktop.py` | dev 모드 launcher (Python subprocess + Edge --app) | 표준 라이브러리 |
| `yoink_main.py` | PyInstaller 패키지 모드 launcher (단일 프로세스) | uvicorn |

### Frontend (단일 HTML)

| 섹션 | 줄 수 | 역할 |
|---|---|---|
| `<style>` | ~1100 | CSS 변수, 컴포넌트별 스타일 |
| `<body>` HTML | ~250 | 헤더, 필터, 모달, popover, bulk-bar 구조 |
| `<script>` JS | ~1700 | 검색, 그리드, 모달, 설정, 가이드, 에러 로그, 학습 추천 |

의도적으로 단일 파일 — 빌드 단계 없음, Ctrl+F5 한 번이면 UI 변경 즉시 반영.

## 데이터 저장

### SQLite (`images.db`, WAL 모드)

```sql
CREATE TABLE images (
    id INTEGER PRIMARY KEY,
    path TEXT UNIQUE,        -- 절대 경로 (resolve 후)
    filename TEXT,
    tags TEXT,               -- AI 태그 (한국어 자유형, ", " 구분)
    description TEXT,        -- AI 한 줄 설명
    clip_embedding BLOB,     -- float32 × 512 (CLIP)
    indexed_at TIMESTAMP,
    -- 마이그레이션 추가:
    mtime REAL,
    folder_id INTEGER,
    wd_chars TEXT,           -- WD14 영문 (","), 캐릭터+작품
    wd_chars_ko TEXT,        -- 한국어 캐릭터 (", "), 사용자 편집 가능
    wd_general TEXT,         -- WD14 일반 영문 ("," 상위 30개)
    hidden INTEGER DEFAULT 0,
    favorite INTEGER DEFAULT 0,
    user_tags TEXT,          -- 사용자 직접 입력 ("," 구분)
    ccip_embedding BLOB      -- float32 × 768 (CCIP, L2 normalized)
);

CREATE TABLE folders (
    id INTEGER PRIMARY KEY,
    path TEXT UNIQUE NOT NULL,
    added_at TIMESTAMP,
    last_indexed_at TIMESTAMP
);

CREATE TABLE char_prototypes (
    name TEXT PRIMARY KEY,   -- 한국어 캐릭터 이름
    centroid BLOB NOT NULL,  -- float32 × 768 (online mean)
    count INTEGER NOT NULL,  -- 누적 라벨 수
    updated_at TIMESTAMP
);
```

### 파일 시스템 보조 저장소

| 경로 | 내용 |
|---|---|
| `thumbnails/{id}_{mtime}_{size}_v2.webp` | 카드 그리드용 480px WebP (애니메이션 보존) |
| `db_backups/images_{ts}_{reason}.db` | 위험 작업 직전 자동 스냅샷 (최근 5개 유지) |
| `.browser_data/` | Edge --app 모드의 user-data-dir |
| `~/.cache/huggingface/hub/` | 모델 다운로드 캐시 (Qwen, WD14, CCIP) |
| `settings.json` | AI provider 설정 (API 키 평문 — 추후 OS keyring 마이그레이션 고려) |
| `character_aliases.json` | WD14 영어 → 한국어 매핑 사전 (사용자 직접 편집 가능) |

## 모델 스택

| 역할 | 모델 | 크기 | 속도 (CPU/GPU) |
|---|---|---|---|
| 시각 임베딩 | CLIP ViT-B/32 (open_clip, laion2b) | ~600MB | CPU 50ms, GPU 5ms |
| 캐릭터 인식 | WD14 EVA02-Large (SmilingWolf) | ~400MB | CPU 500ms |
| 캐릭터 동일성 학습 | CCIP CAFormer-24 (deepghs) | ~150MB | CPU 120ms |
| VLM (로컬 기본) | Qwen2.5-VL-7B-Instruct | 16GB VRAM | GPU 2~5s |
| VLM (로컬 4bit) | + bitsandbytes nf4 quant | 5.5GB VRAM | GPU 3~6s |
| VLM (로컬 작음) | Qwen2.5-VL-3B-Instruct | 7GB VRAM | GPU 1~3s |
| VLM (API) | OpenAI / Anthropic / Gemini | 0 VRAM | API 1~3s + 비용 |
