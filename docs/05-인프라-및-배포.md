# 05 — 인프라 및 배포

## 실행 환경

- **OS**: Windows 10/11 (현재 전용)
- **Python**: 3.13
- **GPU**: NVIDIA (옵션, 없으면 API 모드)
- **브라우저**: Edge 또는 Chrome (`--app` 모드용)
- **인터넷**: 모델 첫 다운로드 시 + API 모드 사용 시

## 3가지 실행 경로

### A. 패키지된 EXE (가장 쉬움, GPU 불필요)

**대상:** Python 모르는 일반 사용자

```
1. Releases에서 Yoink-vX.X.X.zip 다운로드 (~550MB)
2. 압축 풀기
3. Yoink.exe 더블클릭
4. 첫 실행 시 ⚙ 설정 → API 키 입력
5. 폴더 인덱싱 시작
```

- AI 분석은 API 모드만 (Gemini/OpenAI/Anthropic)
- CLIP/WD14/CCIP는 CPU로 동작
- 첫 실행 시 모델 자동 다운로드 (~1GB)
- 데이터: `Yoink.exe` 옆 `images.db`, `settings.json`, `thumbnails/`

### B. 소스에서 — 풀 기능 (개발자 / GPU 사용자)

**대상:** Local Qwen VLM까지 쓰고 싶은 사용자

```powershell
# uv 설치 (아직 없다면)
winget install astral-sh.uv

git clone https://github.com/vanillapapaya/MemeTracker.git
cd MemeTracker

# RTX 5080(Blackwell)이면 nightly torch:
uv pip install --index-url https://download.pytorch.org/whl/nightly/cu128 --pre torch torchvision
# RTX 30/40 시리즈는:
# uv pip install --index-url https://download.pytorch.org/whl/cu124 torch torchvision

uv pip install -r requirements.txt

# (선택) LAN HTTPS — mkcert 인증서
# mkcert -install
# mkcert <LAN-IP> 127.0.0.1 ::1

# 실행
.venv\Scripts\python.exe desktop.py
```

### C. 소스에서 — API 전용 경량 (CPU만)

**대상:** GPU 없지만 API로 로컬과 동등한 품질 원하는 사용자

```powershell
git clone https://github.com/vanillapapaya/MemeTracker.git
cd MemeTracker
uv venv .venv-cpu
$env:VIRTUAL_ENV = "$PWD\.venv-cpu"

uv pip install --index-url https://download.pytorch.org/whl/cpu torch torchvision

               onnxruntime huggingface_hub openai anthropic google-genai numpy

.venv-cpu\Scripts\python.exe app.py
```

⚙ 설정에서 API provider 키 입력.

## 데스크톱 런처 (Edge --app 모드)

### Dev 모드 (`desktop.py`)

흐름:
1. 단일 인스턴스 lock (포트 50815)
2. 백그라운드 스레드에서 `python.exe app.py`를 subprocess로 실행
   - exit 42 → 자동 재실행 (헤더 ↻ 재시작 버튼용)
3. 메인 스레드는 서버 listen 대기 후 Edge/Chrome `--app=<URL>` 띄움
4. 브라우저 닫히면 서버 종료

### 패키지 모드 (`yoink_main.py`)

- subprocess Python 사용 불가 (번들엔 Python 인터프리터 없음)
- 단일 프로세스에서 uvicorn을 background thread로 실행
- Edge/Chrome 찾아 `--app` 띄움
- 환경변수 `YOINK_PORT`, `YOINK_INSTANCE_PORT`로 포트 오버라이드 가능 (테스트용)

### Edge `--app` 옵션
```
--app=<URL>
--user-data-dir=<로컬 폴더>     ← 사용자 일반 Edge와 격리
--window-size=1400,900
--no-first-run
--no-default-browser-check
--disable-features=Translate
```

→ 결과: 진짜 데스크톱 앱처럼 보이는 독립 윈도우 (URL 막대, 탭, 즐겨찾기 없음).

## PyInstaller 패키징

### 빌드 환경

```powershell
uv venv --python 3.13 .venv-build
$env:VIRTUAL_ENV = "$PWD\.venv-build"

# CPU torch 사용 (패키지 크기 + GPU 모드는 .venv에서만)
uv pip install --index-url https://download.pytorch.org/whl/cpu torch torchvision
uv pip install -r requirements.txt pyinstaller
```

### 빌드

```powershell
.\build.bat
# 결과: dist\Yoink\Yoink.exe + _internal/ (~550MB onedir)
```

### `yoink.spec` 구성

| 카테고리 | 값 |
|---|---|
| `datas` | `templates/`, `character_aliases.json`, `icon.ico`, `icon.png` + open_clip / huggingface_hub 데이터 |
| `hiddenimports` | `providers`, `openai`, `anthropic`, `google.genai`, `google.genai.types`, uvicorn 내부, onnxruntime, `PIL._tkinter_finder` |
| `excludes` | `transformers`, `qwen_vl_utils`, `bitsandbytes`, `accelerate`, `torchaudio`, `tkinter`, `matplotlib`, `IPython` |
| `console` | False (GUI 모드, 콘솔 창 없음) |
| `icon` | `icon.ico` |
| 모드 | onedir (onefile은 빌드 매번 풀어 느림) |

### 패키지된 EXE 구조

```
dist/Yoink/
├── Yoink.exe              # 부트로더 + 메인
└── _internal/
    ├── templates/         # Jinja2 템플릿
    ├── character_aliases.json
    ├── icon.ico
    ├── icon.png
    ├── base_library.zip
    ├── *.dll              # MSVCP, torch C 확장 등
    └── ...
```

`_internal/`은 `sys._MEIPASS`로 노출됨. `app.py:_bundle_resource()` 헬퍼가 자동으로 dev/패키지 양쪽 모드 지원.

## LAN HTTPS (선택)

원격 기기(다른 PC, 폰)에서 클립보드 API 사용하려면 HTTPS 필요. self-signed로 충분.

```powershell
# 1회만:
choco install mkcert -y
mkcert -install
mkcert <PC-LAN-IP> 127.0.0.1 ::1
# → <PC-LAN-IP>+2.pem, <PC-LAN-IP>+2-key.pem 생성
```

`app.py`는 시작 시 같은 디렉토리의 `.pem` 자동 감지 → 있으면 HTTPS, 없으면 HTTP.

## 인증서 자동 생성 (향후)

- mkcert 외부 의존 제거 위해 Python `cryptography`로 self-signed 인증서 자동 생성 검토
- 첫 실행 시 1회 생성, `cert.pem`/`key.pem` 캐시
- → 사용자가 인증서 단계 의식할 일 없음

## 백업 & 복원

### 자동 백업

위험한 작업(`relocalize`, `dedupe_tags`, `cleanup_library_dupes`, `auto_translate_chars`) 진입 직전 자동:

```
db_backups/images_20260527-143022_relocalize.db
db_backups/images_20260527-142511_dedupe_tags.db
...
```

- 최근 5개만 유지 (mtime 정렬)
- 사용자가 사고 시 수동으로 백업 파일을 `images.db`로 복사하면 복원

### 수동 백업 권장

- `images.db`
- `settings.json`
- `character_aliases.json`
- (선택) `thumbnails/` — 재생성 가능하지만 시간 절약
